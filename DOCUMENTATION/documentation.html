1. About

MIO is a responsive HTML template for online resumes. The template has two skins, with three color choices and two portfolio layouts. MIO has an ajax contact form for best User Experience .

MIO template has been implemented with bootstrap framework.

2. Files Structure

The main folder contains the following files, and folders:

- index.html
- sendmail.php
- css
- images
- js

3. HTML Structure

The image represents the structure of the html.

4. CSS

The css files are the following:

- style.css : main css by default
- dark.css : the css for black text
- green.css; css for green hover and links
- orange.css: css for orange hover and links

To select the black text uncomment the following line on the index.html file

<link href="css/dark.css" rel="stylesheet" >

To change the color of hover effects and links to green, you can uncomment the following line on the index.html file

<link href="css/green.css" rel="stylesheet">

To change the color of hover effects and links to orange, you can uncomment the following line on the index.html file

<ink href="css/orange.css" rel="stylesheet">

5. Javascript

The files that can be found on the js directory are as follow:

- jquery.js : javascript library used to produce the animations
- html5shiv.js : Source code adds new HTML5 elements.
- jquery.easing.min.js : Used for the navigation effect
- scripts.js : General jquery scripts to produce some of the animation and functionality

6. PHP

The template contains a PHP file which is required to send a message using the contact form.

To set up the contact form you need to open the sendmail.php file and go to line 7:

mail( "mail@website.com", "Request Information",$message, $headers );

and replace mail@website.com with your email.

7. Browsers compatibility

Template is optimized for Chrome, Safari, Firefox, Opera, IE9, IE8, IE7.

In Internet Explorer hover effect on Portfolio doesnt display like the rest of the browsers.
